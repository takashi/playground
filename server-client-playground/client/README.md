# client


client sample

## Build

```shell
git clone
cd ./client
npm install
npm run build
```
Inside, you will run with `jsx --harmony ./app/components/*.jsx > ./app/components/*.js; browserify ./app/app.jsx -o ./public/app.js`

## Dependencies

- react
- express
- fluxxor
- body-parser
- cookie-parser
- morgan
- serve-favicon
- debug
- localforage
